# GraphQL Projects Directory

This directory contains all the projects built using **GraphQL**.
The primary focus of these projects is working with the **Admin GraphQL API from Shopify**.

Most of the implementations in this folder demonstrate how to interact with the Shopify Admin API using GraphQL for operations such as fetching, creating, updating, and managing store data.

### Focus Areas

* Using **Shopify Admin GraphQL API**
* Querying resources like products, orders, customers, and collections
* Working with **metafields**
* Implementing GraphQL queries and mutations
* Learning practical GraphQL usage in e-commerce applications

Each project inside this directory may showcase different use cases, experiments, or learning implementations related to Shopify's GraphQL ecosystem.
